## 🚀 **Getting Started**

### **Prerequisites**
- Node.js 18+
- PNPM package manager
- Modern web browser for development

### **Local Development**

```bash
# Clone the repository
git clone https://github.com/your-username/hit-website-landing-page-minimax.git
cd hit-website-landing-page-minimax

# Install dependencies for landing page
cd hit-landing-page
pnpm install

# Start development server for landing page
pnpm dev

# In a new terminal, install dependencies for app demo
cd ../hit-web-app
pnpm install

# Start development server for app demo
pnpm dev
```

### **Build for Production**

```bash
# Build landing page
cd hit-landing-page
pnpm build

# Build app demo
cd ../hit-web-app
pnpm build

# Built files will be in dist/ directories
```
